# Sudoku Solver

This is a simple Sudoku solver using a backtracking algorithm with a Tkinter-based GUI.

## Files

- `sudoku_solver.py`: Contains the `SudokuSolver` class with the backtracking algorithm.
- `sudoku_gui.py`: Contains the `SudokuGUI` class for the Tkinter GUI.
- `main.py`: The main script to run the application.

## Usage

To run the Sudoku solver, execute the `main.py` file:

```bash
python main.py

Requirements
Python 3.x
Tkinter (usually comes pre-installed with Python)

Features
Interactive Sudoku board where users can input their own puzzles.
Solve button to solve the Sudoku puzzle using the backtracking algorithm.
Restart button to reset the board to the original state.
